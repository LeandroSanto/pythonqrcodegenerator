Pequena aplicação python que gera api_rest para gerar códigos QR.
Essa pequena aplicação utiliza Fask, QRcode[Pil]

Instruções:
antes de executar, instale as seguintes bibliotecas:
  pip install flask
  pip install qrcode
  pip install qrcode[pil]

  python.\qr_api.py
